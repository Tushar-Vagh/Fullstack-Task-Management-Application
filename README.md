# Fullstack Task Management Application

A full-stack Task Management application built using **React (Frontend)** and **Node.js + Express + MySQL (Backend)** with **JWT Authentication** and **CRUD functionality**.

---

## Features

### Authentication
- User Signup
- User Login
- JWT-based Authentication
- Protected Routes
- Change Password

### User Profile
- Fetch user profile
- Update profile details
- Secure account handling

### Task Management
- Create Task
- Get Tasks (user-specific)
- Update Task
- Delete Task
- Task Status: TODO / IN_PROGRESS / DONE

### Frontend
- Responsive UI (Mobile + Desktop)
- Dark / Light Mode
- Animations using Framer Motion
- Tailwind CSS styling

### Backend
- RESTful APIs
- MySQL database
- JWT authorization middleware
- Password hashing using bcrypt
- Auto table creation
